## 功能亮点

- Visual Studio 2022 Watch / Locals / Autos 放大镜查看 `HObject`、`HTuple`、`HDict`。
- 独立 Viewer 支持 VS 变量、本地图片、`.hvx`、`.hobj`、`.tup`、`.hdict` 和文件夹拖拽浏览。
- 图像、区域、XLD 可叠加显示，区域使用固定 LUT12 颜色顺序。
- 支持测量、阈值预览、灰度剖面、直方图、截图复制、标注和文件夹缩略图缓存。
- 普通图片查看可走离线托管通路；VS 变量和 HALCON 原生文件读取需要目标电脑具备 HALCON 环境。
